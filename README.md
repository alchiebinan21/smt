## Installation procedure
- `git clone https://github.com/alchiebinan21/smt.git`
- `cd smt`
- `composer install`
- `npm install`
- `php artisan key:generate`
- `php artisan migrate`
- `php artisan db:seed`
- `php artisan serve [--port=8000]`
- `npm run prod`
